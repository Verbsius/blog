# 22. 括号生成
## QUESTION
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

## SOLUTION
~~~
class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> ans;
        string path(n * 2, 0);
        
        // 目前填了left个左括号，right右括号
        auto dfs = [&](this auto && dfs, int left, int right) -> void {
            if (right == n) {
                ans.push_back(path);
                return;
            }

            // 填左括号
            if (left < n) {
                path[left + right] = '('; //
                dfs(left + 1, right);
            }

            // 填右括号
            if (left > right) {
                path[left + right] = ')';
                dfs(left, right + 1);
            }


        };

        dfs(0, 0);
        return ans;
    }
};
~~~

## 解释
*  因为是直接覆盖，所以不需要恢复现场
